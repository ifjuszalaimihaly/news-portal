# PHP News Portal (No Framework)

This is a PHP news portal created **without using any PHP framework**.

## Key Functionality

- List news
- Read news one by one
- Upload news with:
  - Title
  - Intro
  - Text
  - Published date
  - Image
- Edit news
- Delete news
- Users can edit or delete **only their own** news items
- Users can log in to the system
